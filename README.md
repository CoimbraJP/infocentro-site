# 🌐 Infocentro — Institutional Website

Institutional website and digital portfolio for Infocentro, a computer store and tech repair shop.

---

## 📋 About

Built to present the store's services, make it easy for customers to get in touch, and serve as the business's digital presence. Focused on performance and professional design.

### What it covers

- Services offered (sales, maintenance, technical support)
- Contact information and location
- Portfolio of completed work
- Support and contact channels

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

---

## 🚀 Running locally

```bash
git clone https://github.com/CoimbraJP/infocentro-site.git
cd infocentro-site
npm install
npm run dev
```

Open `http://localhost:3000`
